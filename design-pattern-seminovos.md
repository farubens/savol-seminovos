# Design Pattern — Projeto Site de Seminovos (WP Headless)

## Objetivo do Projeto

Criar um front-end moderno para um site de seminovos utilizando **WordPress Headless como backend**, com um visual que converse diretamente com o **varejo automotivo**, mas mantendo uma estética **bonita, premium e moderna**, sem parecer elitista ou inacessível.

O objetivo visual é transmitir:

> “Comprar carro aqui é simples, confiável, moderno e seguro.”
